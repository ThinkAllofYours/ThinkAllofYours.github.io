## 문제번역(OpenAI)

크기가 각각 `m`과 `n`인 두 정렬된 배열 `nums1`과 `nums2`가 주어졌을 때, 두 배열의 **중앙값**을 반환하세요.

전체 실행 시간 복잡도는 `O(log (m+n))`이어야 합니다.

**예시 1:**

```
입력: nums1 = [1,3], nums2 = [2]
출력: 2.00000
설명: 병합된 배열은 [1,2,3]이고 중앙값은 2입니다.
```

**예시 2:**

```
입력: nums1 = [1,2], nums2 = [3,4]
출력: 2.50000
설명: 병합된 배열은 [1,2,3,4]이고 중앙값은 (2 + 3) / 2 = 2.5입니다.
```

**제약 조건:**

- `nums1.length == m`
- `nums2.length == n`
- `0 <= m <= 1000`
- `0 <= n <= 1000`
- `1 <= m + n <= 2000`
- `106 <= nums1[i], nums2[i] <= 106`

## 풀이법 

처음에 문제를 보자마자 그냥 리스트 두개를 합치고 nums = nums1 + nums2 

nums를 소팅하고 

len(nums)가 짝수이면 예를들어 4면 4/2,  4/2 + 1 에 위치 한 값을 가져오면 되고

len(nums)가 홀수이면 예를들어 5면 3의 위치한 값을 가져오면 됨

🚩해결

```python
from typing import List

class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        nums = nums1 + nums2
        nums.sort()
        if len(nums) % 2 == 0:
            return (nums[len(nums) // 2] + nums[len(nums) // 2 - 1]) / 2
        else:
            return nums[len(nums) // 2]
```
