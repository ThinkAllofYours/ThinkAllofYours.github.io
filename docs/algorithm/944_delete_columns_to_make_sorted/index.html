🚀 944. Delete Columns to Make Sorted

Delete Columns to Make Sorted - LeetCode

간단보이는 문제다.

문제 해석은 strs = ["abc", "bce", "cae"] 이렇게 값이 주어졌을때 | a | b | c | | b | c | e | | c | a | e |

세로로 a b c 는 정렬되어 있고

세로로 b c a 는 정렬되어 있지 않다.

그래서 b c a 인 두번째 열은 지워야 하고 주어진 문자에서 총 몇개를 지워야 하냐?가 문제이다.

그래서 일단 브루트 포스로 풀었다.

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        del_col = 0
        # strs[0][0] strs[1][0] str[2][0]
        length = 0
        if len(strs) > 0:
            length = len(strs[0])
        sorted_col = [[] for _ in range(0, len(strs[0]))]
        for col in range(0, length):
            for i in range(0, len(strs)):
                sorted_col[col].append(strs[i][col])
        for chars in sorted_col:
            if chars != sorted(chars):
                del_col += 1
        return del_col

속도가… 너무 안좋다. 자 더 빠르게 개선을 해보자

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        del_col = 0
        # strs[0][0] strs[1][0] str[2][0]
        col_length = 0
        if len(strs) > 0:
            col_length = len(strs[0])
        for col in range(0, col_length):
            for i in range(0, len(strs)):
                if i - 1 >= 0 and (strs[i][col] < strs[i - 1][col]):
                    del_col += 1
                    break
        return del_col

어차피 한번은 돌아야 하는 for 문 안에서 한방에 끝내려 했다.

만족스럽군..🤔

근데 더 느리다? 메모리 사용양은 확실히 줄었는데 느린게 기분이 나쁘다

Chat GPT에게 리팩토링을 시켰더니 더 깔끔하게 만들어줬다. (리팩토링은 더 잘하는것 같다.)

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        deletions = 0
        for col in range(len(strs[0])):
            for i in range(1, len(strs)):
                if strs[i][col] < strs[i-1][col]:
                    deletions += 1
                    break
        return deletions

속도개선을 물어봤더니

class Solution:
    def minDeletionSize(self, strs: List[str]) -> int:
        deletions = 0
        for col in range(len(strs[0])):
            prev_char = strs[0][col]
            for i in range(1, len(strs)):
                if strs[i][col] < prev_char:
                    deletions += 1
                    break
                prev_char = strs[i][col]
        return deletions

이렇게 만들어줬다.

음… 이건 결국 내 코드랑 속도차이는 크지 않을 것 같다.

그래서 문제를 풀어달라 했다.

그랬더니 풀어준 답은 이것과 같다.

def minDeletionSize(strs):
    deletions = 0
    for col in range(len(strs[0])):
        for i in range(1, len(strs)):
            if strs[i][col] < strs[i-1][col]:
                deletions += 1
                break
    return deletions

위에 리팩토링한 답과 같다.
